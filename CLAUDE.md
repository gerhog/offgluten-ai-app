# CLAUDE.md

This is an early-stage project for the Offgluten AI app.

## Workflow Rules

- Do not install dependencies unless explicitly asked
- Do not initialize git unless explicitly asked
- Keep changes small and controlled
- If you disagree with any part of a prompt or plan, make the case before starting — name what and why, suggest an alternative
- Explain the plan before making changes
- Summarize exactly what changed after each task
- Ask before acting when requirements are ambiguous
- If breaking a task into subtasks would improve the work, do it without asking
