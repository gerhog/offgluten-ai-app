# Offgluten AI App

AI-powered gluten-free assistant. Early-stage skeleton, ready for Vercel deployment.

## Stack

- Next.js 15 (App Router)
- TypeScript
- ESLint

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Routes

| Path | Description |
|------|-------------|
| `/` | Home page |
| `/login` | Login (placeholder) |
| `/app` | Main app (placeholder) |
| `/chat` | AI chat (placeholder) |

## Deploy

Push to GitHub and import into [Vercel](https://vercel.com). No additional configuration needed.
